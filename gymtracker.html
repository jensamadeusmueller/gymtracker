Install on Phone
1️⃣
Host on GitHub Pages
Create a free GitHub repo → upload this file as index.html → enable GitHub Pages in repo Settings → Pages
2️⃣
Open on your phone
Visit https://yourname.github.io/gymtracker in Safari (iOS) or Chrome (Android)
3️⃣
Add to Home Screen
iOS: Share → "Add to Home Screen"  |  Android: Menu → "Install app" — works offline, looks like a native app
⚠️

Data is stored in your browser's localStorage and persists indefinitely — but will be lost if you clear browser data. Export a backup regularly to stay safe.
